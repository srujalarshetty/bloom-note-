# 🌸 Bloom Note

**Bloom Note** is an aesthetic web application that allows users to create a personalized bouquet with a meaningful note and share it with others.

Instead of using cluttered images, Bloom Note focuses on **clean, minimal flower elements arranged beautifully inside a vase**, creating a soft and visually pleasing experience.

---

## ✨ Features

* 🌿 **Flower Selection**

  * Choose from flowers like Rose, Tulip, Sunflower, and Cherry Blossom
  * Add flowers to your bouquet with a single click

* 🏺 **Aesthetic Bouquet Builder**

  * Flowers are arranged in a structured, clustered layout
  * No messy overlapping—clean positioning using CSS

* 🎨 **Vase Customization**

  * Select different vase styles (Glass, Ceramic, Minimal)
  * Instantly updates the bouquet display

* 📝 **Personal Note**

  * Add a meaningful message to your bouquet
  * Displayed beautifully below the arrangement

* 🔗 **Shareable Link**

  * Generate a link to share your bouquet
  * Others can view the exact same bloom

* 📱 **Responsive Design**

  * Works smoothly across desktop, tablet, and mobile

## 🚀 How It Works

1. Select flowers from the grid
2. Flowers are added to a structured bouquet layout
3. Choose a vase style
4. Add a personal note
5. Share your bloom using a generated link
